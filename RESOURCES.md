# ASP.NET Core Resources

## Knowledge

- [Video: "Clean Architecture with ASP.NET Core 10" — dotnet (.NET Conf 2025)](https://www.youtube.com/watch?v=rjefnUC9Z90)
  Official .NET team session on Clean Architecture principles, project organization, and refactoring toward loosely-coupled code. Primary resource for architecture concepts.
- [GitHub: jasontaylordev/CleanArchitecture](https://github.com/jasontaylordev/CleanArchitecture)
  The most popular Clean Architecture solution template for ASP.NET Core (20k+ stars). The reference pattern our OmniBlox structure is based on.
- [Article: "Clean Architecture in ASP.NET Core" — NDepend Blog](https://blog.ndepend.com/clean-architecture-for-asp-net-core-solution/)
  Thorough breakdown of the four layers (Domain, Application, Infrastructure, Presentation) with coupling analysis.
- [Article: "Vertical Slice Architecture in ASP.NET Core" — Code Maze](https://code-maze.com/vertical-slice-architecture-aspnet-core/)
  Explains the difference between layered (Clean) and vertical slice architecture with MediatR.
- [Docs: "Entity Framework Core — Getting Started" — Microsoft](https://learn.microsoft.com/en-us/ef/core/)
  Official EF Core documentation. Use as primary reference for all DbContext, migrations, and Fluent API questions.
- [Docs: "FluentValidation in ASP.NET Core" — Microsoft](https://learn.microsoft.com/en-us/aspnet/core/fundamentals/validation)
  Official docs on request validation patterns, including FluentValidation integration.
