# Prior knowledge: C# basics, no ASP.NET experience

The user has some C# basics — comfortable with syntax, classes, and basic programming constructs, but has never built an ASP.NET Core web API before. They come from a TypeScript/Next.js/NestJS background.

**Implications:** Lessons should not re-teach C# fundamentals (loops, classes, async/await) but must explain every ASP.NET-specific concept from first principles. Analogies to NestJS concepts (DI, controllers, guards, interceptors) will be effective bridges.
